NeoTaylor Shorthand

A modern modified Taylor shorthand

NeoTaylor is a geometric heavy-line abjad shorthand designed to be easy to pick up and start using. It is based on Samuel Taylor's shorthand published in 1786. However NeoTaylor introduces many additions to the system including shading, updated brief forms, punctuation new rules, and new strokes.

If you wish to contribute to the website or share your own example writtings then you can check out the GitHub repository or contact me at "ljcottendev@gmail.com".
